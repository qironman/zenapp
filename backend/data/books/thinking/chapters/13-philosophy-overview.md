# Philosophy

## Summary

### Ancient Philosophy (c. 600 BCE – 500 CE)

| Era             | Branch * School | Key Concepts * Thinkers                            |
|-----------------|-----------------|----------------------------------------------------|
| Ancient Greek   | Pre-Socratic    | Naturalism, arche (Thales, Heraclitus, Pythagoras) |
|                 | Socratic        | Ethics, dialectics (Socrates, Sophists)            |
|                 | Platonism       | Theory of Forms, reason, the Good (Plato)          |
|                 | Aristotelianism | Logic, empiricism, virtue ethics (Aristotle)       |
| Hellenistic     | Stoicism        | Virtue, rational order, apatheia (Zeno)            |
|                 | Epicureanism    | Pleasure, atomism, ataraxia (Epicurus)             |
|                 | Skepticism      | Suspension of judgment (Pyrrho, Sextus Empiricus)  |
| Late Antique    | Neoplatonism    | The One, emanation, mysticism (Plotinus)           |
| Ancient Indian  | Vedanta         | Non-dualism, Brahman, Atman                        |
|                 | Samkhya         | Dualism of consciousness and matter                |
|                 | Buddhism        | No-self, Four Noble Truths, emptiness (Buddha)     |
|                 | Jainism         | Anekantavada, ahimsa                               |
| Ancient Chinese | Confucianism    | Social harmony, rites, ren                         |
|                 | Daoism          | Naturalness, wu wei, Dao                           |
|                 | Mohism          | Universal love, meritocracy                        |
|                 | Legalism        | Law, discipline, centralized rule                  |

### Medieval Philosophy (c. 500 – 1500)

| Era            | Branch * School    | Key Concepts * Thinkers                      |
|----------------|--------------------|----------------------------------------------|
| Christian West | Patristics         | Faith and reason, grace (Augustine)          |
|                | Scholasticism      | Faith | Aristotelian logic (Aquinas, Anselm) |
| Islamic        | Peripatetic School | Reason and revelation (Avicenna, Averroes)   |
|                | Illuminationism    | Mystical light metaphysics (Suhrawardi)      |
| Jewish         | Jewish Rationalism | Negative theology, ethics (Maimonides)       |

### Early Modern Philosophy (c. 1500 – 1800)

| Era          | Branch * School    | Key Concepts * Thinkers                                  |
|--------------|--------------------|----------------------------------------------------------|
| Early Modern | Rationalism        | Innate ideas, deductive reason (Descartes, Leibniz)      |
|              | Empiricism         | Experience-based knowledge (Locke, Hume, Berkeley)       |
|              | Social Contract    | Political order from agreement (Hobbes, Locke, Rousseau) |
|              | Deontology         | Duty, categorical imperative (Kant)                      |
|              | Natural Philosophy | Scientific method (Galileo, Newton, Bacon)               |

### 19th-Century Philosophy (c. 1800 – 1900)
| Era          | Branch * School | Key Concepts * Thinkers                               |
|--------------|-----------------|-------------------------------------------------------|
| 19th Century | German Idealism | Reality via mind, dialectic (Kant, Hegel, Fichte)     |
|              | Utilitarianism  | Happiness, consequentialism (Bentham, Mill)           |
|              | Existentialism  | Authenticity, freedom, angst (Kierkegaard, Nietzsche) |
|              | Marxism         | Historical materialism, class struggle (Marx)         |
|              | Positivism      | Empirical science as truth (Comte)                    |
### 20th & Contemporary Philosophy
| Era            | Branch * School      | Key Concepts * Thinkers                              |
|----------------|----------------------|------------------------------------------------------|
| Analytic       | Logical Positivism   | Verification, anti-metaphysics (Carnap, Ayer)        |
|                | Ordinary Language    | Language in use, clarity (Wittgenstein, Austin)      |
|                | Philosophy of Mind   | Consciousness, identity (Ryle, Dennett, Chalmers)    |
|                | Epistemology/Science | Falsifiability, paradigm shift (Popper, Kuhn)        |
|                | Political Philosophy | Justice theory, libertarianism (Rawls, Nozick)       |
| Continental    | Phenomenology        | Conscious experience (Husserl, Heidegger)            |
|                | Existentialism       | Absurdity, choice, alienation (Sartre, Camus)        |
|                | Structuralism        | Underlying structures in thought (Lévi-Strauss)      |
|                | Post-structuralism   | Power, discourse, deconstruction (Foucault, Derrida) |
|                | Critical Theory      | Culture, ideology critique (Adorno, Habermas)        |
| Eastern Modern | Neo-Vedanta          | Hindu synthesis with modernity (Vivekananda)         |
|                | Engaged Buddhism     | Ethics, social activism (Thich Nhat Hanh)            |
|                | New Confucianism     | Modern revival of Confucian values                   |
