# 摩洛哥之旅 古惑仔摩洛哥的那些事
今天离开Fez向南部进发。中午参观了一个法国人留下的葡萄酒庄园，在优雅的泳池旁尝了酒，吃了一顿法国菜。然后参观了酿酒的全过程。参观完等着被兜售葡萄酒，却得知老板不知何故失去了卖酒执照。这尼玛也太悲催了，前戏作这么充分，马上高潮到来，我都准备好掏银子带两瓶回去，他却不能卖！不禁想起了甄嬛里学的“对食”，问君能有几多愁，恰似一群太监上青楼。又好像中国男足，尽管举国之力高薪支持，关键时刻临门一脚永远是雄起的反义词。
由于路途遥远，路上导游讲了一路摩洛哥的近代史，我的脑海里却浮现出郑伊健的英雄形象。 为啥尼？19世纪初，欧洲大陆由于第一次工业革命出现了一群肌肉男，块头大了不出去欺负人有点憋得难受，所以英法西班牙这几个流氓大哥开始全世界到处耍流氓占地盘。当时摩洛哥南北部被西班牙占领，中部被法国占领。法国为啥占中部尼？因为中部矿产资源丰富，收的保护费更高些。保护费不是白收，老大帮着小弟修了很多公路铁路啥的。后来爆发了两次黑帮大火并，史称两次世界大战。在第二次世界大火并中，摩洛哥毅然决然站在了大哥法国一边，送了上万兄弟去欧洲战场，旗帜鲜明地打击江湖崛起的大流氓纳粹德国。有人可能问为啥要帮欺负了你几十年的流氓法国？敌人的敌人是朋友，为啥不帮德国打法国？ good question. 我认为心理学家已经给出答案了，那就是斯德哥尔摩综合症--一群心理学家研究了一个奇怪现象，就是被劫持的人质经常会出现同情劫匪的心理，甚至竭力帮助劫匪。史上另外一个例子就是小日本二战被美国用两个原子弹强奸蹂躏，之后还奴颜婢膝贱人般的的一直帮着大哥美国作事。
古惑仔摩洛哥不是白白帮法国大哥砍人流血的，大哥讲义气，答应灭了德国之后把地盘还给摩洛哥。答应是答应，可纳粹投降后法国可没让摩洛哥立马独立，为啥？二战火并很多男丁都被砍死，战后重建严重缺少劳动力，所以大哥从殖民地大抽血，抓了无数壮丁猪仔回法国。几年后摩洛哥才独立。塞翁失马焉知非福，这些猪仔及后人都成为法国公民。想当初跑到台湾的国民党反动派后来都成为富商回祖国大陆投资包二奶，留守大陆的基本被无三反五反文革啥的给整死。
摩洛哥虽然独立了但是南北大部分土地还是被另一个纳粹流氓西班牙占领。话说西班牙已经收了摩洛哥几百年的保护费了。你就是唱一百遍“一个芝麻糕不是我真姓”人家也不还你，直到七十年代。
这段历史我印象中的中学历史课本讲了一小段，就是二战后亚非拉美的独立运动，还是选修内容。我就不明白当代世界格局形成这么大的事为啥是选修内容。到了美国一对比发现天朝的历史教法不大对路，基本上就是叉叉年叉叉把叉叉打服了，收了保护费，历史意义是以下十条叉叉，很少问为什么，what if。 美国这边更多的是鼓励学生思考，多问为什么，如果没有叉叉会发生什么叉叉。这才是学历史真正的意义。这亚非拉战前被欧洲列强瓜分，战后殖民者人和资源拼的差不多了，要重建自顾还不瑕，那殖民地当然有机会独立了。就机会你也得自己给力争取，不要以为战后这些殖民统治者会突然良心发现，改邪归正把地盘归还。香港澳门一直都是殖民地，直到世纪末才回归祖国。
话说古惑仔摩洛哥七几年终于从西班牙手里收回大部分土地，只剩下北方两个城市人家就是不还。西班牙的理由很简单很粗暴：啥时候英国这厮把直布罗陀还给俺，俺就把这两个城还你。这种流氓手段有木有让大家想起刘备诸葛亮赖着荆州不还的故事？等哥们把西川取了再还你荆州。呵呵，西川取了还是不还你。耍赖的结果是成全了《大军师司马懿》，人家魏吴联盟水淹七军，你就败走麦城。
虽然被殖民多年，小弟下面还有小弟，摩洛哥南面有一群非洲小部落联盟，在独立后这些小弟也产生了想法。靠，你可以独立，为啥俺不能？老子也不当小弟了。南面的黑非洲国家纷纷起哄表示支持。可惜，虽然摩洛哥派了大批部队多年清剿反政府游击队，由于没有诸葛亮这样七擒孟获的特殊人才，最后这些小屁部落居然独立了。
所以古惑仔摩洛哥告诉我们，“人在江湖飘，怎能不挨刀”。”出来混，总是要还的”。
