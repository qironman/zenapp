# 摩洛哥之旅 沙漠宿营

尽管有些水土不服，我们还是坚持跟着队伍继续向沙漠进发。上午到达一个村子，进了一个摩洛哥大厨的庄园学习烧摩洛哥菜。
大厨首先介绍了摩洛哥的沏茶工序。摩洛哥的传统茶没有从亚洲进口的绿茶，只包括薄荷艾蒿和几种土生土长植物。自从臭名昭著的东印度公司把亚洲的绿茶带到欧洲和非洲，摩洛哥的茶又多了一味绿茶。这位号称得了泡茶精髓的摩洛哥老者虽然也认识到泡茶要慢工出细活，但是他们慢是为了大家聊天唠嗑。而且懂泡茶的都知道绿茶不宜用开水泡，更不能一煮半天。这和我们的茶道比差远了。我们的茶道讲的是修心养性，茶具，水和茶叶都是很讲究的，那是道。这老者说他们泡茶必须用不锈钢的壶当时我差点没笑喷了。比这个再土一点的也就是当年的罐头瓶子沏茶了吧。茶道咱不大懂，陆羽的茶经也没看全。其实就是排除杂念，剩下的就两件事：放茶叶，倒水。水的温度很重要，绿茶绝不能放开水，更不能放二糊涂水，要烧开后冷却到华氏180左右。红茶大概190。放茶顺序一般是先放茶再倒水，泡毛尖儿一定要后放茶叶，不然水会显得浑浊。
然后大厨开始详细讲解如果烹制摩洛哥闻名于世的Tagine. 看了一会儿感觉这不就是砂锅炖X吗？X可以是羊肉鸡肉或是各种蔬菜。这个我就不得不赞一下我们伟大祖国丰富多彩的饮食文化了。煎炒烹炸，色香味意形。即然你摩洛哥大厨口口不绝的说你们的食材和香料都是本地山上出产，那我们先不跟你提我们的八大菜系，鸳鸯五珍烩，黄蓉的二十四桥明月夜，就是我们土特产的菜也让你瞠目结舌。历史上有名的丐帮帮主洪七公爱吃的叫花鸡，还有石破天在摩天涯烧的獐肉羹，俘获了谢烟客的芳心。
