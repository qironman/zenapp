# 君子固穷
这年头，“逆来顺受”都快升级成标配了，像胸牌，像工号，像二维码——不扫不让进。你问他想要啥？他一乐：有也行，没有也行，都行。听着像修行，细看像省事；省的不是世界的事，省的是自己的事，省到最后，连“不甘心”这仨字都嫌费电。
这种人最讲“体面”。不吵不闹，不争不抢，像一团面，生活一揉再揉，揉到没筋儿了，还能自我发明一条口头禅：随便。老板扣他绩效，他说随便；房租涨得比火箭还直，他说随便；别人拿走他的时间、尊严、机会，他还是随便。随便得跟腰里别着一部《道德经》似的——仿佛穷不是穷，是低调的富；庸不是庸，是高级的淡；挨打不是挨打，是“我不跟你一般见识”。
可他嘴上说“都能接受”，心里那小算盘啪啦啪啦响得可热闹：既然改变太麻烦，那就把失败写成哲学；既然拿不到糖，那就宣布自己戒糖；既然站不起来，那就说“我喜欢躺着”。于是现代版阿Q就上线了：他不反抗，他反解释；他不翻身，他翻白眼；他不赢，他先把“输”定义成“另一种赢”。在平庸里找尊严，在贫穷里找富裕，在委屈里找大度——这不是“君子固穷”，这是把穷活成一套自我麻醉的礼仪，端着，敬着，还得自己给自己鼓掌。
“君子固穷”原来是什么？是硬气，是骨头，是明知路窄也不跪着走。到了今天呢，硬气常常被换成软话：别较真，别折腾，别给自己添堵。堵在哪儿？堵在嗓子眼儿，堵在胃里，堵在眼皮底下，堵在那口“算了”的气里。人一旦把“接受”当美德，把“忍耐”当智慧，就很容易把“被按着头”当成“自愿低头”，最后连谁摁的都忘了，只剩一个姿势——端端正正地受着，规规矩矩地受着，还要总结经验：你看，我多成熟。
所以我说，这类“现代君子”，看着清醒，其实糊涂；看着通透，其实圆滑；看着淡定，其实怕事。他们不怕穷，他们怕麻烦；他们不求好，他们求“过得去”。过得去，过得去，过得去——过得去久了，连“过不去”是什么滋味都忘了。只剩下一句：有也行，没有也行。听着云淡风轻，实际上是把命交出去，还装作自己在掌舵；明明船早让人牵着走，他还站在甲板上挥手：我这叫随缘。随什么缘？随别人的安排，随自己的胆怯，随那句最省事的——随便。
