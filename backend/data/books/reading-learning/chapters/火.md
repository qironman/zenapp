# 火

人类在发明火之前是不是都茹毛饮血？别急着点头，你先想想你今天中午要是端上来一盘生肉，你敢不敢当着大家的面嚼得跟健身博主似的，还顺带说一句“原汁原味”？你不敢。你最多敢在朋友圈里敢。
说白了，生的东西就一个劲儿——就那股“这玩意儿本来就这样”的味儿。肉是肉，花生是花生，硬邦邦、寡淡淡，嚼半天跟嚼人生似的：费劲，没回报。可你一把火给它招呼上，嘿，世界一下就有戏了。肉一烤，香味就像突然学会了说话，立刻把你从“我吃饱了”哄成“再来一串”。花生一烤，更别提，外头焦黄一点，里头油脂一冒，香得跟谁家孩子考上大学似的，满屋子都跟着高兴。
这就是能量的魔法——大自然的东西本来就在那里，平平无奇，你给它加一点外来的劲儿，像给人上了点发条，它就开始发光发热，身价立刻翻倍。可问题也在这儿：火这玩意儿有脾气，给少了不行，给多了更不行。火候不到，还是那股“本来就这样”；火候过了，直接糊给你看——糊了就别谈什么“风味”，那叫“报废”。人这一辈子，最怕的就是把火候当成了态度：不是一股劲儿猛烤，就是死活不敢点火，结果要么糊，要么生，谁也别笑谁。
你再往大里看，火其实就是小太阳。咱们都活在这个太阳系里，地球这点小日子，靠的就是太阳天天照着、烘着、养着。没有它那点能量润着，你别说文明了，你连“原始”都原不起来——冻得你连石头都懒得捡。于是人类一看：哎哟，这东西这么厉害，那不拜一下说得过去吗？埃及人弄个太阳神，波斯人拜火，火在地上烧，太阳在天上烧，一个是分店一个是总店。后来你看谁谁谁的故事里也总爱来点光、来点火、来点圣焰，毕竟大家心里都明白：你得有点能量，才能把日子烤熟。
再往根上说，没有太阳，就没有光合作用；没有光合作用，植物不长，动物不肥；不长不肥，你吃什么？吃空气吗？更别提什么煤啊油啊那些埋在地下的宝贝——那都是远古的阳光攒出来的欠条。太阳要是不照，你连欠条都没得写。你看，人类折腾来折腾去，最后不过是学会了两件事：第一，火得点；第二，火得掌握火候。别把自己烤糊了，也别把日子过成生的。
