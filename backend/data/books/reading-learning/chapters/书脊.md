# 书脊
书的脊。
在很久以前，书并不一定有“脊”。竹简一片片串起，卷舒之间，留下的是绳结与缝隙；后来纸成了册，蝴蝶装、包背装、线装，都是把页与页拢在一起，却把“背面”藏起来：它面向墙，面向箱笼，面向尘土。那时人读书，看的是封面与内页，书的背影只负责沉默。
直到书开始站起来。
当书被一排排立在架上，脊才忽然有了名字、有了职责：它要承重，要记名，要在你转头的一瞬间，替你把一本书从一群书里叫出来。书名、作者、编号、馆藏印——都写在那条窄窄的背上。书的脸可以华丽，书的脊必须清醒。
于是你会发现，脊有圆的，也有方的。
圆的，是怕隔手吗？也许最先不是为了手，而是为了骨头。手工装订的厚书，纸张与线绳把书背勒得紧，工匠会把它一下一下捶成弧形，再“翻边”做出肩位，让书壳咬住书体——圆背更能分散拉力，开合时不至于把线扯断。圆背像一段弓，受力时懂得退让；也像人的脊柱，有节、有韧，有一点倔强的弯。
方的，常常更直接。薄册、胶装、平装，书脊像一面小墙，整齐地站在货架上，标题印得笔直，像给时间钉上坐标。方背的书打开时更容易摊平，读起来像把桌面铺开；但胶水老了会脆，纸张多了会裂，它的挺拔里也藏着一种易碎。
你问“隔手”，其实问的是人与书相处的方式。圆背握在掌心里，有微微的抵抗，你得顺着它的弧度读，像顺着一个人的呼吸；方背更像工具，利落地被抽出、插回，被折角、被划线，被塞进包里又被压在雨伞下面。一个更像“陪伴”，一个更像“使用”。当然这只是偏见：有些圆背冷硬，有些方背温柔，书的性情不全写在外形上。
但书脊总是诚实的——它记录了它被打开过多少次，被合上过多少次，被谁的指腹捏出过油光，被谁的手汗浸出过暗斑。它把阅读的重量悄悄扛在身上，把人的时间压成一条线。
书的脊，是书的背影，也是书的骨。你从脊上认出一本书，就像从背影认出一个人：不必看脸，也知道那里面装着怎样的故事。圆的，方的，都只是形；真正让它不隔手的，是它在你掌心里留下的那一点点温度——像历史，硬着，旧着，却仍然能被握住。
