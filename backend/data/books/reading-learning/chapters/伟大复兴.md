# 伟大复兴

没有比较就没有伤害——这话俗得像路边摊的塑料袋，可一拎起来，里面还真装着沉甸甸的现实。
我刚从国内转了一圈回来，心里“咯噔”一下：原来“震撼”这种词不光能用在风景上，也能用在效率上，用在秩序上，用在一群人把日子过得像个样子上。然后我落回美国，才几天，就被航班延误、取消按在地上摩擦了N次。空管局那套陈旧设备，像是从博物馆里抬出来的；一场屁大点儿的雨，就能把机场整到半身不遂，瘫着躺几天，还躺得理直气壮。更妙的是，这种半瘫痪据说已经持续三周多了——你说它是故障吧，它稳定；你说它是稳定吧，它又总在坏。想起上个月新泽西铁路机师大曝光之前，火车也是频频延误、取消车次，像是大家约好了：能不把你送到，就尽量别把你送到。空管局员工是不是也用消极怠工来抗议？如果真是，那这抗议的对象是谁呢？反正挨刀的总不是他们。
航班一取消，我去查火车，九个小时。九个小时是什么概念？在国内，这距离高铁三小时就把你端到目的地，你还能顺便刷两集剧，打个盹，再醒来假装自己没坐过车。这里九小时，你坐到腰酸背痛，脑子里开始跟自己谈人生：我到底为什么要在这儿体验“时间的厚重”。想租车异地还车？行啊，多掏四百刀。四百刀是什么概念？这钱够你北京上海高铁舒舒服服来回好几趟，车厢里那空调吹得像正义一样均匀。
再说吃的。美国的外卖，贵得像交保护费，难吃得像在接受惩罚，选择还有限得像你活在某种配给年代。你点来点去，最后发现自己不是在选菜，是在选一种更不难受的难受。反观国内，美团一打开，满屏都是“人间值得”：热的、冷的、辣的、甜的、清淡的、重口的，甚至顶花带刺的小黄瓜都能给你送到门口。关键是还便宜，便宜到你突然开始同情商家：这么干，他们靠什么活？你看，这才叫把人逼出一点良心。
于是我坐在延误通知的海洋里，往华盛顿方向望去——那地方据说是希望的制造厂——结果什么也看不见，只看见一片雾，雾里还有指示牌：请耐心等待。基础设施落后得像停在几十年前的老胶片，股市币市却不断刷历史新高，像一个人裤腰带已经断了，还要把领带打得更漂亮。大家都在涨，唯独路不通、车不走、飞机不上天。你说这算什么繁荣？这像极了某种荒诞的体面：外面镀金，里面空心。
我忍不住想：什么时候会有个孩子突然高喊——你们别装了，那位“美帝”其实没穿底裤？可孩子大概早就学会了沉默，学会了在延误队伍里刷手机，学会了把荒唐当常态，把常态当命运。
而我，刚刚从另一边回来，眼睁睁看见有人把路修通，把车跑快，把日子过顺。于是这句口号一样的结论，就这么不情愿却又清清楚楚地冒出来：中国民族真的伟大复兴了。至于这边嘛——衰落这个词太大，我不敢随便扣；但我可以确定一件小事：你站在雨里等飞机的那一刻，你会明白，原来“帝国”也会怕一场小雨。
