# Beliz Trip
## Been There Done That
在伯利兹的大部分时间，我们都待在一条租来的渔船上。海是蓝的，太阳是狠的，我们是晕的。人到了这种地方，通常会产生一种错觉：自己离文明很远，其实离晒伤更近。
第一天去码头，见一小船，名曰 Sweet Jenny。阿甘粉丝都懂这个名字：那个永远跑在命运前面的男人，和那个永远跑在他心里的女人。只不过这艘 Jenny 是玛雅版的——船头绑着褪色的彩绳，像某种海边的祭祀遗留物；船身上有手画的纹样，线条粗得像用宿醉的手指蘸着油漆抹出来的图腾；旁边还贴着一枚半脱落的贴纸，上面是个咧嘴的美洲豹。你看着它，会突然明白：在这儿，浪不是浪，是神的呼吸；而“Sweet”这词儿，也许只是对现实的一点礼貌。
我们租的船，名字更狠：Been there, done that。翻译过来，大概是：没有哥没去过的地儿，没有哥没干过的事儿。或者：没有哥没去过的远方。哥就是传说，哥就是沧海。简言之，老炮儿。
等你看见船老大的身材，就知道这自信牛逼爆棚的船名是哪来的了。怎么形容呢？就是身长七尺，身宽也是七尺。不是夸张，是一种几何事实。他站在甲板上，甲板显得有点心虚。那感觉有点像《Moana》里面的 Maui：你还没问他能不能带你出海，他已经用全身告诉你——当然能，而且海要是不同意，他就跟海谈谈。然后他真的会笑一下。那种笑很简单：世界爱咋咋地。他说Sit back and relax 
船老大这身板子，不仅可以压住船，还可以辟邪，大有宝塔镇河妖之势。
我们出发。于是我们也就出发了。就是这样。
