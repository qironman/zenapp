# 不亦说乎
学而时习之，不亦说乎。
我头一回见这句，差点以为孔子在给人递烟：学习够累了吧？来，喘口气，说说话，吐吐槽，跟同学把心里那点火气放出来，像给脑子开窗通风。不然闷着，脑子先发霉，后学的东西也跟着变味。
然而老师说：少来这套，你这叫胡扯。
于是老师像个熟练的外科大夫，把一把叫“通假”的刀从白大褂里抽出来，哗啦一下，干净利落地插进课本的肉里。通假。重点当然在那个“假”字。假，假的，临时的，凑合的。听着挺谦虚，实际上挺霸道：你别管它合不合理，反正它就这么定了。我的印象里，这不就是古人的错别字吗？只不过古人写错了，隔了两千年，我们得负责记住它为什么错、怎么错、错得多么“有文化”。
他们写竹简，写得慢，写得累，写错一个字，后果却很现代：能变成一代又一代学生的记忆负担。你背得越熟，越显得你尊敬传统；你背得越疼，越显得你认真学习。教育是一台特别会自我证明的机器：它把你的疼痛换成分数，把分数换成更大的疼痛，然后还要你对着镜子说“我成长了”。
那到底是什么“说”？
老师告诉你：不是说话的说，是悦。高兴。喜悦。你懂吗？你得高兴。你要被规定高兴。你不高兴？那就是你态度有问题。你态度有问题？那就再做两套题，做到你笑为止。
可孔子要的，也许不是那种“你考了第一所以你很悦”的悦。那种悦太像奖状的油光：贴在墙上，远看挺亮，近看一股胶味。他要的可能是另一种——你终于明白了某个东西，明白得像突然看见灯亮了一样，那一瞬间你会笑一下，甚至想对旁边的人说一句：“原来是这样。”这才像“说”，也才配叫“悦”。这句里的快乐，不是别人发给你的，是你自己从黑暗里捞出来的。
当然，课堂上更常见的快乐是：下课铃响了。那铃声比“悦”诚实。
我不怪老师。老师也被训练过。他们也在反刍。他们也背过“通假”。他们也曾经在某个深夜，对着一堆古人写错的字发过誓：我一定要把这些错背得比正确还正确。然后他们长大了，成了老师，把誓言兑现给我们——一人一份，不许退货。
所以我现在再读“学而时习之，不亦说乎”，会稍微宽容一点，也稍微尖刻一点。我会想：孔子也许是在说——别把学习当成挨打。学一点，回头再嚼一嚼，嚼到你能说出来，嚼到你能跟别人聊一聊，嚼到你能笑一下。你笑那一下，不是因为你赢了谁，而是因为你终于没被那些“假”字吓住，没被那台机器逼着假装高兴。把反刍当快乐，痛并快乐的，这就已经很不容易了。
