# 铁匠1996年的第一场雪
Aerosmith’s Cryin
第一次听这首歌，是在1995年的第一场雪里。穷学生兜里没钱，脑子里有梦，手里抱着从中关村淘来的盗版Win95光盘，神情像捧着诺贝尔奖，心情像偷到未来。
Aerosmith一开口我就服了：这不是嘴，这是舞台设备；这不是嘴角，这是地理边界；这不是唱歌，这是声带起义。毫不夸张地说，那嘴一张，五个包子算前菜。什么叫天赋异禀，什么叫老天追着喂饭，什么叫原装扩音器？嘴大到这个级别，还拿来只说“在吗”，那就是暴殄天物，外加资源错配。
Aerosmith被译成“空中铁匠”，我一边拍案叫绝，一边替英语老师叹气。人家最初想叫Arrowsmith，嫌土，改成Aerosmith。smith是“匠”，locksmith是锁匠，可铁匠偏偏是blacksmith。把这事翻成“空中铁匠”，像把西装改成秋裤，保暖是保暖，气质也确实接地气。
那年代电脑是奢侈品，正版是传说，穷书生的音乐来源主要靠三样：盗版、打口、北京音乐台。无数冷得发抖或热得冒烟的夜晚，《零点乐话》是北漂学子的深夜食堂。伍洲彤大哥，您不是主持人，您是情绪急救箱；您不是知心大姐，您是午夜民政局。可音乐台有些栏目也真猛，反复播歌像执行任务，播到前奏响三秒，听众集体反胃。花心、吻别、情网，轮流轰炸。等刁寒《花好月圆》横空出世，我当场怀疑这词是把整条街的吉祥话搅拌后倒出来的：每句都似曾相识，连起来不知所云。讲真，这属于九十年代人肉版AI，抄得有胆，缝得有劲，后人模仿多年，始终学不会那股理直气壮。
一眨眼快30年过去。我带着“好人一生平安”的朴素愿望去搜伍哥近况，得知您娶了演员刘冯合欢。先高兴，再沉思，最后确认我的祝福有点多余：这名字本身就把婚姻生活写成了剧透，既“合欢”，又喜庆，还自带相声节奏。此处只好借一句老台词收尾：娶妻如此，夫复何求。#英文金曲[话题]#
