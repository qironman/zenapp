# 铁匠1995的第一场雪
Aerosmith《Cryin》
那年我在中关村抱回一张盗版 Win95 光盘，抱得跟抱新生儿似的，轻拿轻放，生怕一抖把青春抖成马赛克。盘里有个小视频，点开一看是 Aerosmith。主唱一开口，我当场肃然起敬：这哪是嘴，这是天窗；这哪是唱歌，这是开闸；这哪是器官，这是老天爷亲手发的扩音喇叭。毫不夸张地说，五个包子排着队都能和平进驻。嘴大到这份上，再让它只负责“吃饭说话”，那就叫暴殄天物，属于对自然资源的浪费。
这支伟大的乐队被译成“空中铁匠”，我一听就服，服得五体投地还带打滑。洋名里有个 smith，就给人家安排成铁匠，逻辑之猛，像把英语词典当炒菜谱用。其实 locksmith 是锁匠，blacksmith 才是铁匠。把 Aerosmith 直接锻造成“空中铁匠”，说明翻译老师可能不是教英语的，是教立定跳远的，起跳有力，落点感人。
那年月，电脑是奢侈品，正版更是奢侈品里的奢侈品。我们这些穷书生的音乐来源很朴素：盗版、打口、北京音乐台。冷天听，热天听，失恋听，假装没失恋也听。《零点乐话》一响，多少北漂学子跟着电波续命。伍洲彤大哥，您那会儿就是午夜情感办主任，选歌有温度，有分寸，有人味儿。反观某些栏目，拿几首俗歌循环轰炸，今天放、明天放、后天还放，放到前奏三秒，听众就条件反射想找塑料袋。典型曲目：花心、吻别、情网。后来刁寒《花好月圆》横空出世，我听得眼前一亮又一黑：每句都似曾相识，合起来云山雾罩，像把十本歌词本撕碎了拿风扇一吹，落哪句唱哪句。要按今天的话说，这属于九十年代的人肉生成式模型，原创性主要靠勇气支撑。
一眨眼快三十年。带着“好人一生平安”的朴素愿望，我去搜了搜伍哥近况。看到您娶了演员刘冯合欢，我先祝福，再感慨：这名字起得太讲究，前半句像春联，后半句像谜底，既合欢又喜庆，听着就有相声包袱的回响。此情此景，只能借唐伯虎一句老话收尾：娶妻如此，夫复何求。#英文金曲[话题]#
